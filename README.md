# Sistema de Biblioteca de Mángas Simples

## O que ele faz?
Esse programa é uma biblioteca simples de mangás feito em C# para fins de estudo, aprendizado e prática onde você pode pegar emprestado um mangá de 5 opções já pré-definidas, além de poder tambem devolver mangás que voce pegou da lista de mangás diponíveis.

Além dessa funcionalidades a biblioteca também contém funções secundárias como: 

-Sistema de Pesquisa em que voce pode pesquisar pelos mangás dentro da lista

-Estatisticas que mostram a porcentagem de mangás disponíveis e emprestados.


## Qual o motivo para a criação deste projeto?
Como dito anteriormente, esse projeto tem apenas o objetivo de fazer parte do meu aprendizado de C# e Pensamento Computacional. Neste programa eu usei conceitos como, variáveis, vetores, loops, condicionais, operadores(aritméticos, atribuição, comparação e lógicos) e entrada e saída de dados.

## Pré-requisitos(Para rodar pelo Terminal)
- C# (via .NET SDK 9.0. ou superior)
- Terminal ou qualquer prompt de comando
